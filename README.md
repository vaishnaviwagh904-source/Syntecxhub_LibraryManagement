# Library Management System

A simple Java console-based Library Management System.

## Features

* Add new books
* Remove books
* Search books
* Display all books

## Technologies Used

* Java
* ArrayList
* Scanner

## How to Run

1. Compile the code:
   javac LibraryManagement.java

2. Run the program:
   java LibraryManagement

## Project Description

This project is created as part of the Syntecxhub Internship Week 1 Task.
It helps manage books in a library using a command-line interface.
